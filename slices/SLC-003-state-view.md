# SLC-003 State View

## Slice ID
SLC-003

## Name
State View

## Related Feature
FEAT-001 Project Overview

## Purpose
Render the most important current-state information for the selected project.

## Goal
Display the selected project's current operational state clearly.

## Included in scope
- project name
- delivery mode
- high-level state
- current phase
- current focus
- next steps
- blockers
- last stable version

## Explicitly out of scope
- full timeline history
- editing all state data directly
- analytics

## Expected result
The user can open the overview and quickly understand where the project currently stands.

## Acceptance criteria
- state information is clearly visible
- next steps are visible
- blockers are visible
- state visibility improves resume speed

## Risks
- unclear mapping from source records
- cluttered overview presentation

## Priority
high

## Status
planned
