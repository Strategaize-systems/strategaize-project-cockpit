# SLC-008 Decisions View

## Slice ID
SLC-008

## Name
Decisions View

## Related Feature
FEAT-006 Decisions / Skill Improvements View

## Purpose
Render project decisions and dev-system improvement findings in a structured view.

## Goal
Make important technical and process learnings visible for review and reuse.

## Included in scope
- decision record rendering
- skill improvement record rendering
- identifier or section visibility
- short rationale visibility
- simple readable structure

## Explicitly out of scope
- automated evolution workflows
- approval engine
- direct editing of all records in V1

## Expected result
The user can review what decisions shaped the project and what repeated learnings should improve the system.

## Acceptance criteria
- decisions are visible
- skill improvements are visible
- review is easier than raw file browsing

## Risks
- mixing too many concepts into one view
- weak source structure

## Priority
medium

## Status
planned
