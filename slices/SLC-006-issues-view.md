# SLC-006 Issues View

## Slice ID
SLC-006

## Name
Issues View

## Related Feature
FEAT-004 Bugs / Issues View

## Purpose
Render known issue and bug information in a structured UI.

## Goal
Provide a visible issue layer so open problems are not hidden.

## Included in scope
- issue list rendering
- identifier visibility
- issue title
- status
- short symptom summary
- severity or priority where available

## Explicitly out of scope
- full issue management workflows
- assignment system
- comment system
- SLA handling

## Expected result
Known issues are visible enough to support reliable review and release awareness.

## Acceptance criteria
- issue records render clearly
- open issues are visible
- issue review is easier than raw file inspection

## Risks
- inconsistent issue data structure in source records
- overbuilding issue logic in V1

## Priority
medium

## Status
planned
