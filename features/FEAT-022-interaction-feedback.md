# FEAT-022 — Interaction & Feedback States

## Status
planned

## Priority
medium

## Description
Konsistente Interaction-States und Feedback-Mechanismen auf allen Elementen gemäß Style Guide.

## In Scope
- Hover: Cards (elevated + Y-2px), Buttons (elevated + Y-1px), Rows (gradient + left-border)
- Focus: 3px Brand-Color Ring auf allen interaktiven Elementen
- Disabled: Grayed out, no-pointer, 0.6 opacity
- Loading: Brand-Color Spinner (ersetze Text-Loading)
- Toast-Notifications: Bottom-Right, Border-Left nach Typ
- Animation-Timing: 200ms Micro, 300ms Cards, 500ms Progress

## Out of Scope
- Skeleton Loading States
- Complex Animations
