# FEAT-023 — Page-spezifische Upgrades

## Status
planned

## Priority
medium

## Description
Seiten-spezifische visuelle Verbesserungen die nicht durch die generischen Komponenten-Upgrades abgedeckt werden.

## In Scope
- Übersicht: Header-Banner mit Brand-Gradient (Blau → Indigo) + weißem Text
- Releases: Timeline-Stil mit farbigen Dots + expandierbare Details
- Roadmap: Meilenstein-Timeline mit Fortschrittsbalken
- Nächster Schritt: Dunkler Prompt-Block + großer Kopieren-Button + priorisierte Aufgabenliste
- Backlog: Sprint/Story-Point-Darstellung (visuell)
- Reports: Category-Tags + Author-Spalte (visuell, Daten wenn vorhanden)

## Out of Scope
- Neue Datenfelder hinzufügen (nur visuelles Upgrade bestehender Daten)
- Neue Seiten
