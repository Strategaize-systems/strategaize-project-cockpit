# FEAT-020 — Premium Tabellen

## Status
planned

## Priority
high

## Description
Tabellen-Upgrade: Gradient-Top-Border, sortierbare Header mit Hover, Row-Hover mit Gradient und Left-Border, Action-Icons bei Hover, Status-Dots, Fortschrittsbalken.

## In Scope
- 4px Gradient-Top-Border (Brand-Gradient)
- Header: Uppercase, Sortier-Hover (Brand-Color)
- Row-Hover: Subtle Gradient-Background + 4px Left-Border + ID in Brand-Color
- Action-Icons: Hover-Only, max 4, View/Edit/Delete-Hover-Colors
- Status-Dots neben Titeln (mit Glow-Effect)
- Fortschrittsbalken in Slices-Tabelle
- Anwendung auf: Features, Slices, Backlog, Reports

## Out of Scope
- Spalten hinzufügen/entfernen
- Tatsächliche Sortier-Funktionalität (nur visueller Header-State)
